URL Shortener API

A simple service that returns a JSON object containing the original URL and a shortened URL.

Example Input:

https://dry-plains-62025.herokuapp.com/api/https://www.youtube.com/watch?v=D-UmfqFjpl0

Example Output:

{ original_url: "https://www.youtube.com/watch?v=D-UmfqFjpl0", short_url: "https://dry-plains-62025.herokuapp.com/1003" }
